# Screenshots

Add your Power BI dashboard screenshots here after building the dashboards.

Suggested filenames:
- `01_overview_dashboard.png`
- `02_company_analysis.png`
- `03_timeline_trends.png`
- `04_interview_analytics.png`
- `05_resume_skills.png`

To take screenshots in Power BI:
1. Go to each dashboard page
2. Press `Windows + Shift + S` (Windows snip tool)
3. Save as PNG and drop into this folder
4. Then run: `git add screenshots/ && git commit -m "add dashboard screenshots"`
